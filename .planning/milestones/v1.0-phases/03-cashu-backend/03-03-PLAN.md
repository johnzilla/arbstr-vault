After querying `auditEntries`, find the PAYMENT_REQUEST entry and extract routing metadata:
```typescript
const paymentRequestAudit = auditEntries.find(
  (e) => (e.action as string) === 'PAYMENT_REQUEST'
);
// auditLog stores metadata as JSON text — need to query the full row
```

Actually — the current query only selects `action` and `created_at` from auditLog. Extend it to also select `metadata`:
```typescript
const auditEntries = (db as unknown as NarrowDb)
  .select({
    action: auditLog.action,
    created_at: auditLog.created_at,
    metadata: auditLog.metadata,
  })
  .from(auditLog)
  .where(
    and(
      eq(auditLog.agent_id, id),
      eq(auditLog.ref_id, tx_id),
    ),
  )
  .all();
```

Then extract routing fields from PAYMENT_REQUEST metadata:
```typescript
const paymentRequestEntry = auditEntries.find(
  (e) => (e.action as string) === 'PAYMENT_REQUEST'
);
const routingMeta = paymentRequestEntry?.metadata as Record<string, unknown> | null;

const initialRail = routingMeta?.initial_rail as string | undefined;
const finalRail = routingMeta?.final_rail as string | undefined;
const fallbackOccurred = routingMeta?.fallback_occurred as boolean | undefined;
```
